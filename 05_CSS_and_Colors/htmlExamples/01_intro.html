Getting Started with CSS

Rules

CSS works based on rules

The rules are usually specified in a file. The file is often called "style.css" but other names work as well.

Selectors

HTML elements may be used as selectors.

IDs may be used as selectors. IDs must start with # in the .css file.

Syntax

CSS rules do not use html syntax.
